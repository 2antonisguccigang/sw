   Τα αρχεία δεν αποτελούν τον καλύτερο τρόπο οργάνωσης την πληροφορίας, καθώς με τους περιορισμούς που έχουν στα ονόματα για παράδειγμα και στον χώρο αποθήκευσης, μελλοντικά      δυσκολεύουν την κατασταση.
