με αποτέλεσμα να κατηγορούν την τεχνολογία. Για αυτό και θεωρεί οτι πρέπει να εξετάζουμε όλες αυτές τις μεθόδους και όλα
αυτά τα προϊόντα που εμφανίζονται. Διαφωνεί με την άποψη κάποιων που δεν διαφωνούν με την τεχνολογία. Τονίζει πως δεν πρέπει
να παραπλανούμαστε απο την συσκευασία καθώς δεν είναι αυτή η τεχνολογία. Δίνει για παράδειγμα τα iPhone κινητά. Εξηγεί οτι
η τεχνολογία δεν βρίσκεται σε αυτό αλλά στην συσκευασία του ή αλλιώς στο πακετάρισμα. Επόμενο παράδειγμα είναι το Microsoft
Windows όπου υπάρχει πακετάρισμα και τεχνολογία. Το Macintosh που είναι πακετάρισμα και τεχνολογία. Το wrapper όμως θεωρεί

τρόπος οργάνωσης της πληροφορίας απο την πλευρά του χρήστη καθώς ο χώρος εξαντλείται στα ονόματα καταλόγου ή στον λαβύρινθο

εφιάλτης των χρηστών είναι η ασυμβατότητα των ονομάτων των αρχείων μεταξύ του λειτουργικού συστήματος και αυτό πολλές φορές
οφείλεται στους απαγορευμένους χαρακτήρες. Τέλος υπάρχουν δείκτες που μπορεί να οδηγούν κάπου που δεν γνωρίζουμε.
