# Άσκηση Γραμμής Εντολών 2

mqtt= πρακτικό πρωτόκολλο για την διαχείριση μηνυμάτων. Μπορεί να κάνει publish μηνύματα. Αν κάποιος ενδιαφέρεται να διαβάσει αυτά τα μηνύματα και να ενημερωθεί με αυτά τα μηνύματα, μπορούν να πάνε να κάνουν εγγραφή. Στην συνέχεια τους έρχεται ένα email και λαμβάνουν αντίστοιχα μια ειδοποίηση. Πρόκειται για μια εύχρηστη εφαρμογή καθώς διαχωρίζει τα μηνύματα σε topics. Συνεπώς μπορεί κανείς να έχει έναν mqtt ο οποίος να δέχεται από διάφορες πηγές μηνύματα τα οποία πηγαίνουν σε διαφορετικά σημεία. Καθένας μπορεί να ακολουθεί ότι τον ενδιαφέρει. Είναι πολύ ελαφρύ πρόγραμμα και για να απολαύσουμε όλα αυτά που μας προσφέρει θα πρέπει να κάνουμε τα εξής: αρχικά κάνουμε την εγκατάσταση του με την εντολή sudo apt-get install mosquitto και ελέγχω αν τρέχει /etc/init.d/mosquitto . Για να δω σε τι κατάσταση βρίσκεται sudo /etc/init.d/mosquitto status σε περίπτωση που δεν τρέχει το ενεργοποιώ με την εντολή sudo /etc/init.d/mosquitto start και κάνω ξανά έλεγχο. Στην συνέχεια κάνω εγκατάσταση του client με την εντολή sudo apt-get install mosquitto-clients . Στέλνω μηνύματα με το mosquitto_pub -m “Hello world from mosquitto_pub client” -t “notifications” ενώ ταυτόχρονα το ενεργοποιώ. Κάνω εγγραφή με το mosquitto_sub -t “notifications” & . Στην συνέχεια εκτελώ τις εντολές mosquitto_pub -m “Hello world #2 from mosquitto_pub client” -t “notifications” . Τώρα όποτε υπάρχει ένα μήνυμα θα εμφανίζεται. Μετά κάνω εγκατάσταση του mqttwarn με sudo pip3 install --upgrade mqttwarn , φτιάχνω φάκελο mkdir mqttwarn και ακολουθώ τα βήματα που μου λέει στο site του mqttwarn και φτιάχνω τα αρχεία και τα εμφανίζω με ls. Για να δω που συνδέθηκε και που έκανε εγγραφή πατάω την εντολή mqttwarn. Τέλος με την εντολή vi mqttwarn.ini βλέπω που βρήκε τα διάφορα topics που έχει κάνει εγγραφή, επεξεργάζομαι αυτά τα δεδομένα και συνδέω και άλλα όπως το http , το file και άλλα.
