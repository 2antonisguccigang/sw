# Συμμετοχικό Περιεχόμενο 1

Σε αυτό το στάδιο πρόσθεσα δυο φωτογραφίες με ελεύθερα πνευματικά δικαιώματα. Η μια μικρή πλάτους 160πίξελ και μια κανονική. Το θέμα που επέλεξα είναι το λογισμικό Microsoft Cortana. Ένας εικονικός βοηθός που αναπτύχθηκε από τη Microsoft για την εκτέλεση εργασιών όπως η ρύθμιση υπενθυμίσεων και η απάντηση ερωτήσεων από τον χρήστη. Η αναζήτηση έγινε μετά απο την αντίστοιχη ρύθμιση στις φωτογραφίες του google. Στην συνέχεια δημιούργησα ένα χρονολόγιο που συνδυάζει τις δύο νέες εικόνες με εικόνες που ήδη υπάρχουν.
