## <a name="dk01">DoKey - Send Notifications:</a>

Στο παρών παραδοτέο κατέβασα και προσάρμοσα το ntfy στα υπολογιστή μου.  Εκμεταλλεύτηκα το ntfy ώστε να δημιουργήσω ένα script που κάθε φόρα που ανοίγω τον υπολογιστή μου θα με ειδοποιεί αν υπάρχει κάποιο νέο update. Κάτι πολύ χρήσιμο μιας και δεν είχα κάποιο πρόγραμμα που να κάνει την αντίστοιχη δουλεία στο λειτουργικό μου, και συχνά κατέληγα να μένω πίσω στα updates και να έχω διάφορα προβλήματα (όπως να μην λειτουργούν νέα πακέτα).

Επειδή το checkupdates χρειάζεται ίντερνετ για να δουλέψει και υπάρχει ένα μικρό χρονικό διάστημα από την στιγμή που θα ανοίξει ο υπολογιστής το οποία ο  δεν έχει συνδεθεί ακόμη στο ίντερνετ είναι απαραίτητη η πρώτη while. Στην συνέχεια αν υπάρχουν πακέτα για αναβάθμιση έρχεται ειδοποίηση στον χρήστη αλλιώς δεν συμβαίνει τίποτα.
```
  while ! ping 8.8.8.8 -c 1; do sleep 0.5; done
   
  n=$(checkupdates | wc -l)
   if [ "$n" != "0" ]; then
    ntfy -t "System update" send "$n packages for update available."
  fi
```
#### Παράδειγμα:
![image review](https://i.ibb.co/9sZbj1n/2021-03-15-18-25.png)

Επίσης το γεγονός ότι το ntfy σου δίνει την δυνατότητα να καλέσεις ειδοποίηση μόλις εκτελεστεί μια εντολή, μου έδωσε την ιδέα να δημιουργήσω ένα δεύτερο script το οποίο θα καλώ όταν θέλω να κάνω upgrade.  Επειδή η διαδικασία του upgrade καμία φορά μπορεί να είναι αρκετά χρονοβόρα θεώρησα χρήσιμο να μου έρχεται  μια ειδοποιήσει μόλις ολοκληρώνεται.

Στο δεύτερο script είναι πρακτικά μόνο μια εντολή αλλά προτίμησα να την βάλω σε αρχείο ως script και να κάνω το script alias.  ( Ώστε να είναι πιο εύκολο  να κάνω μελλοντικές αλλαγές)
```
  ntfy -t "System update" done sudo pacman -Syyu python
```

Να μην περιοριζόμαστε με τις τεχνικές προγραμματισμού που υπάρχουν αλλά να αφήνουμε την δημιουργικότητά μας “ελεύθερη”.  Με απλά λόγια να μην πιστεύουμε ότι “βρήκαμε την λύση” στον προγραμματισμό. |
