Python Scripts:

###### rand_ints.py

```python
import random
import sys

n = sys.argv[1] #Number of random numbers to generate
afile = open(n+".txt", "w" )
print("Producing random numbers...")

for i in range(int(n)):
    line = str(random.randint(1, 10000))
    afile.write(line)
    afile.write("\n")

afile.close()
print("done")
```
