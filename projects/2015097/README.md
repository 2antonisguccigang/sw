# Εισαγωγή

Μέσα απο τη συμμετοχή μου στη διδασκαλία του μαθήματος έχουν προκύψει στόχοι σχετικοί με το μάθημα τους οποίους θα ήθελα να εκπληρώσω, αλλα και ανάγκες οι οποίες θα ήθελα να ικανοποιηθούν. Ως πρώτο στόχο θέτω να αποκτήσω όλα τα κατάλληλα εφόδια που θα με βοηθήσουν να ανταπεξέλθω στις ανάγκες του μαθήματος. Ως δεύτερο στόχο θέτω να λάβω πολλές γνώσεις σχετικά με το μάθημα Τεχνολογία Λογισμικού, καθώς είναι ενα πολύ ενδιαφέρον μάθημα που τίθεται να προσφέρει πολλαπλά εφόδια σε οποιοδήποτε φοιτητή το παρακολουθήσει.Ως τρίτο στόχο θέτω να είμαι συνεπής με όλες τις παραδόσεις που απαιτεί το μάθημα και να συμμετέχω ενεργά στις διαδικασίες του μαθήματος. Ως τελευταίο στόχο θέτω να την καταβολή προσπάθειας έτσι ώστε τα παραδοτέα που θα υποβληθούν κατά τη διάρκεια του εξαμήνου να είναι όσο πιο τελειοποιημένα γίνεται απο πλευράς μου. Η μόνη ανάγκη που έχω από το μάθημα σας και κατ' επέκταση απο εσάς, είναι ακριβώς η ίδια με το προηγούμενο εξάμηνο: να συνεχίσετε να μας προσφέρετε γνώση ανιδιοτελώς όπως κάνετε τόσο καιρό και να συνεχίσετε να προσφέρεται διδασκαλία υψηλού επιπέδου η οποία εκτιμάται ιδιαίτερα απο φοιτητές οι οποίοι ενδιαφέρονται πραγματικά για την κατάκτηση των γνώσεων που τους προσφέρονται.

# Άσκηση γραμμής εντολών 3

Για την εκπόνηση της τρίτης άσκησης γραμμής εντολών αξιοποίησα το huginn το οποίο χρησιμοποιώντας Agents ενημερώνει το χρήστη για οποιεσδήποτε αλλαγές σε ιστοσελίδες της επιλογής του. Μετά τη προσεκτική μελέτη του documentation δημιούργησα και ρύθμισα εναν website agent ο οποίος κάθε φορά που δημοσιεύεται μια ανακοίνωση στην ιστοσελίδα του Ιονίου Παναεπιστημίου θα με ενημερώνει μέσω ενός δεύτερου agent(post) στο κανάλι μου στο slack.

Πιό αναλυτικά στην αρχή άνοιξα το τερματικό της εικονικής μηχανής μου και εκτέλεσα την εντολή docker run -it -p 3000:3000 --name ionio-sw-lab-huginn huginn/huginn
Στην συνέχεια μέσω του browser του Host λογισμικού (Xubuntu) εισήλθα στην επιφάνεια διεπαφής χρήστη του Huginn από τη διεύθυνηση : http://localhost:3000/ χρησιμοποιώντας τα στοιχεία σύνδεσης που δόθηκαν απο τον διδάσκοντα στο εργαστήριο.
Ύστερα προσέθεσα έναν Agent τύπου Website και τον παραμετροποίησα ως εξής:
```
{
 "expected_update_period_in_days": "2",
 "url": "https://ionio.gr/gr/news/all-news-f1-all-f2-all/",
 "type": "html",
 "mode": "on_change",
 "extract": {
  "url": {
   "xpath": "//*[@class=\"news-list-block\"]/a/@href",
   "value": "."
   },
  "title": {
   "xpath": "//*[@class=\"news-list-block\"]/a",
   "value": "normalize-space(.)"
   }
  }
 }
 ```
 Εκτελόντας το huginn agent με τη μέθοδο Dry Run έλαβα τα ακόλουθα:
 ```
[
 {
 "url": "https://ionio.gr/gr/news/20465",
 "title": "ΒΥΡΩΝ: Η Ελληνική Επανάσταση «ζωντανεύει» με ένα παιχνίδι!"
 },
 {
  "url": "https://ionio.gr/gr/news/20464",
  "title": "Μνημόνιο συνεργασίας μεταξύ του Ιονίου Πανεπιστημίου και του Συνδέσμου Ημερήσιων Περιφερειακών Εφημερίδων"
  },
...
```
Eν κατακλείδι, δημιούργησα έναν ακόμα Agent τύπου Post ώστε κάθε φορά που δημιοσιεύεται κάτι καινούριο στα νέα της ιστοσελίδας του Πανεπιστημίου να ενημερωνομαι μέσω του slack channel, τον οποίο παραμμετροποίησα ως εξής:

```
{
 "post_url": "https://hooks.slack.com/services/********/*******/",
 "expected_receive_period_in_days": "1",
 "content_type": "json",
 "method": "post",
 "headers": {
   "Content-type": "application/json"
  },
 "payload": {
   "text": "{{ title }}; more on {{url}}"
```

- Σύνοψη

Φτάνοντας στη συγγραφή του τελικού κειμένου για την αναφορά μου, θεωρείται εποικοδομητική η εξέταση των στόχων που είχαν γραφτεί στην εισαγωγή και η αξιολόγηση της προσπάθειας μου σε σχέση με αυτούς. Ο στόχος που είχα θέσει για απόκτηση γνώσης πάνω στο θέμα του μαθήματος πιστεύω εκπληρώθηκε, καθώς πλέον έχω εξοικειωθεί αρκετά με την έννοια της Τεχνολογίας Λογισμικού, καθώς και τη φιλοσοφία που τη διέπει. Πηγή αυτών των γνώσεων αποτέλεσαν αφενός οι δυο μορφές video quiz και αφετέρου οι διαλέξεις του μαθήματος.Όσο αφορά το στόχο μου περί συνέπειας εκπόνησης και παράδοσης των ζητουμένων πιστεύω ότι έκανα μια πάρα πολύ δουλειά. Παρέδωσα όλα τα ασύγχρονα βίντεο quiz στην ώρα τους, συμμετείχα σε όλα τα σύγχρονα quiz και παρέδωσα όλα τα παραδοτέα στην ώρα τους (εκτός απο το αίτημα ενσωμάτωσης στην ιστοσελίδα 2 και την άσκηση γραμμής εντολών 4 την οποία έκανα). Θεωρώ λοιπόν, ότι ήμουν συνεπής σε πολύ μεγάλο ποσοστό. Ήμουν αρκετά ενεργός σε όλες τις διαδικασίες του μαθήματος, καθώς και παρόν στον χώρο του mumble και irc. Επιπλέον θεωρώ ότι κατέκτησα τον στόχο μου για απόκτηση χρήσιμων εφοδίων καθώς δούλεψα περαιτέρω πάνω στο github και στο terminal, γεγονός που μου απέδωσε πολύτιμη εμπειρία πάνω σε αυτούς τους τομείς. Εν κατακλείδι, πιστεύω ότι η προσπάθεια που κατέβαλλα άξιζε τον κόπο και το αποτέλεσμα που προέκυψε είναι αρκετά ποιοτικό.
